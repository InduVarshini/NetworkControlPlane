# Scripts

Utility scripts for NetworkControlPlane.

## Available Scripts

- **docker-run.sh** - Helper script to run NetworkControlPlane in Docker
  ```bash
  ./scripts/docker-run.sh
  ```

- **push-to-dockerhub.sh** - Build and push Docker image to Docker Hub
  ```bash
  ./scripts/push-to-dockerhub.sh YOUR_DOCKERHUB_USERNAME
  ```

- **start-ovs.sh** - Start Open vSwitch (required for Mininet switches)
  ```bash
  ./scripts/start-ovs.sh
  ```

## Usage

All scripts should be run from the project root directory.
